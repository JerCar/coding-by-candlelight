# Coding by Candlelight

Static landing page for https://codingbycandlelight.ca

## Run locally

Open `index.html` directly in a browser, or from this folder run:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Deploy

This site contains no build step. DigitalOcean App Platform can publish the repository as a Static Site using `/` as the output directory.
